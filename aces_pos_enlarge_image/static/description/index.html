Odoo 14 POS Enlarge Image (Community)

Allows user to see large product image from POS interface.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • We have one features available in POS Configuration as per below image.


  • After selecting icon we can see large image of product with product name.
